# بطاقات NFC للمكفوفين

مشروع `Node.js + Express` لعمل بطاقة تعريف ذكية مرتبطة بـ `NFC Card 13.56MHz`.
الكارت لا يخزن كل البيانات، بل يخزن رابطًا فقط. عند لمس الكارت بالموبايل تفتح صفحة تعرض:

- الاسم
- العنوان
- رقم الموبايل
- رقم طوارئ
- وصف الحالة
- تسجيل صوتي يشرح التشخيص أو طريقة التعامل

## ما الموجود في المشروع

- صفحة رئيسية تعرض البطاقات الحالية
- صفحة عامة لكل بطاقة على المسار `/p/:publicCode`
- لوحة إدارة على `/admin`
- إضافة وتعديل وحذف البطاقات
- رفع تسجيل صوتي لكل بطاقة
- حماية بسيطة للوحة الإدارة باسم مستخدم وكلمة مرور

## التشغيل محليًا

```bash
npm install
npm start
```

بعد التشغيل:

- الصفحة الرئيسية: `http://localhost:3000`
- لوحة الإدارة: `http://localhost:3000/admin`
- بطاقة تجريبية: `http://localhost:3000/p/demo-card`

في التشغيل المحلي فقط، لو لم تضبط `SUPABASE_URL` و `SUPABASE_SERVICE_ROLE_KEY` فالمشروع يستخدم:

- `SQLite` محليًا
- مجلد `uploads/` لحفظ الصوت

## مسار النشر المجاني

النشر المجاني المقترح للمشروع هو:

- الاستضافة: `Vercel`
- قاعدة البيانات والتخزين: `Supabase`

السبب:

- استضافة Vercel المجانية لا تصلح لحفظ ملفات وصوت و`SQLite` محليًا بشكل دائم
- Supabase يحل مشكلة قاعدة البيانات والتخزين الدائم مجانًا في البداية
- Express مدعوم على Vercel بدون إعدادات معقدة

## إعداد Supabase

1. أنشئ مشروعًا جديدًا على Supabase
2. افتح `SQL Editor`
3. شغّل الملف:

```text
supabase/schema.sql
```

4. من إعدادات المشروع انسخ:

- `Project URL`
- `service_role key`

5. استخدم نفس اسم الـ bucket الافتراضي:

```text
audio-recordings
```

## متغيرات البيئة المطلوبة للنشر

راجع الملف:

```text
.env.example
```

أهم القيم في Vercel:

- `ADMIN_USERNAME`
- `ADMIN_PASSWORD`
- `ADMIN_COOKIE_SECRET`
- `PUBLIC_BASE_URL`
- `SUPABASE_URL`
- `SUPABASE_SERVICE_ROLE_KEY`
- `SUPABASE_STORAGE_BUCKET`
- `MAX_AUDIO_FILE_SIZE_MB`

## ملاحظة مهمة عن الصوت على Vercel المجاني

رفع الصوت يمر عبر السيرفر، لذلك يفضل أن يكون الملف صغيرًا.
الحد الآمن الحالي في هذا المشروع هو:

```text
4 MB
```

إذا أردت رفع ملفات أكبر لاحقًا، نطوّر المشروع إلى رفع مباشر من المتصفح إلى التخزين بدل المرور على Vercel Function.

## ربط كارت الـ NFC

بعد النشر، اكتب على الكارت رابط الصفحة العامة فقط، مثل:

```text
https://your-project.vercel.app/p/card-001
```

نوع السجل على الكارت يكون `URL / URI` داخل `NDEF`.

لا تكتب `localhost` على الكارت، لأن الموبايل سيفتح نفسه بدل الموقع.

## ملاحظات أمان

- لوحة الإدارة يجب أن تبقى محمية بكلمة مرور قوية
- لا تضع بيانات طبية شديدة الحساسية في الصفحة العامة
- من الأفضل لاحقًا إضافة صلاحيات أكثر تفصيلًا وسجل عمليات
